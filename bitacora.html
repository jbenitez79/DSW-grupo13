Bitácora

Descripción del proceso de desarrollo del proyecto.


Decisiones de diseño tomadas por el equipo:

  • Elegir una paleta de colores clara y tipografías modernas para mejorar la legibilidad y la estética.
  • Implementar un menú de navegación horizontal y hacerlo colapsable en dispositivos móviles para mejorar la experiencia de usuario.
  • Separar los estilos CSS y los archivos JS por integrante para facilitar el trabajo colaborativo y evitar conflictos.

Dificultades encontradas y cómo se resolvieron:

  • Problemas de compatibilidad con el menú hamburguesa en dispositivos móviles: se resolvió ajustando los media queries y revisando el JavaScript de apertura/cierre.
  • Dificultades para centrar y alinear correctamente los elementos en diferentes tamaños de pantalla: se solucionó utilizando Flexbox y revisando los márgenes y paddings.
  • Conflictos de estilos entre los archivos CSS de cada integrante: se acordó modularizar el CSS.

Cambios importantes realizados durante la implementación:

  • Se reorganizó la estructura de carpetas para separar imágenes, estilos y scripts, facilitando el mantenimiento.
